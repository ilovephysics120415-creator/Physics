ElectroStatic Hazards

Understanding Electrostatic Hazards

Static electricity builds up when electric charges accumulate on an insulating surface under dry conditions, preventing charges from flowing away. When these charges suddenly discharge, a spark is created. Near flammable materials, this spark can cause fires or explosions!

Select a scenario to begin. Tap "Simulate Charge Buildup" to see static charge accumulate.

Summary of Hazard Reductions:

  • Earthing: Provides a low-resistance path for charges to flow safely to the ground.
  • Conducting Materials: Replaces insulating components to allow constant discharge.
  • Increasing Humidity: Makes air more conductive so charge leaks away continuously.
  • Anti-Static Tools: Prevents charge accumulation on surfaces (e.g., wrist straps).
